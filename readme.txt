This plugin will give you option to give vote on your post through Facebook, twitter, Linkedin and Email.
Admin can approve or delete any vote form admin panel.
Approved votes count will display on each post repectively.


== Description ==
Wp Social Voting plugin will give your visitor to vote on any post in your blog. Voting will be 100% pure as plugin approve it after get proper response from social media.

In this plugin we include Facebook, Twitter, Linkedin and Email voting.

Admin can check the voting from admin panel and delete any approved vote or approve any vote from backend.

There is a plugin setting page where admin have to enter appropriate data to run the voting properly.

Voting functionality is available in widget and can be display before or after the post content, all settings are under plugin setting page.


== Installation ==
1. Upload \"test-plugin.php\" to the \"/wp-content/plugins/\" directory.
1. Activate the plugin through the \"Plugins\" menu in WordPress.
1. Place \"do_action( \'plugin_name_hook\' );\" in your templates.
